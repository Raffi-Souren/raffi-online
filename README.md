# raffi.computer

Personal portfolio site for Raffi Khatchadourian — Field CTO and Technology Leader, DJ, and entrepreneur based in Brooklyn, NY.

Built as a fully interactive Windows XP-inspired desktop environment.

## Live Site

**[raffi.computer](https://raffi.computer)**

## Tech Stack

- **Framework:** Next.js 14 (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **Hosting:** Vercel

## Local Development

```bash
npm install
npm run dev
```

Opens at [localhost:3000](http://localhost:3000).
